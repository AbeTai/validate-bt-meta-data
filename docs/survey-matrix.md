# AVRCP メタデータ調査マトリクス

## 調査項目
| 項目 | 説明 |
|------|------|
| サービス名 | コンテンツ再生サービス |
| アプリ / Web | ネイティブアプリ or ブラウザ経由 |
| 端末 | iPhone, Mac, Windows, Android |
| OS | iOS 18, macOS Sequoia, Windows 11, Android 15 等 |
| BG再生 | バックグラウンド再生中にメタデータが送信されるか |
